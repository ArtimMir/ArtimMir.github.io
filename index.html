Руководство по использованию программы.

1. Выберите режим "Обычное число" или "Комплексное число" в раскрывающемся списке после заголовка "Вычисление корней"

2. Введите число, из которого хотите извлечь корень в поле с надписью "Введите число".

Если выбран режим "Комплексное число", дополнительно введите число перед i.

3. Введите точность, с которой нужно вычислить квадратный корень этого числа в поле с надписью "Введите точность".

4. Нажмите кнопку "Вычислить". Результат будет показан выше кнопки.


Если Вы хотите изменить язык интерфейса, можно сделать это в левом верхнем углу.
